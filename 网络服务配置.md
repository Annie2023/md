# DHCP服务器和客户机的配置

基本步骤书上有，这里写一点注意事项

win2012和win10需要在同一网段，本次使用虚拟机VMnet1（将主机默认的DHCP配置取消掉，这样才能配置成功）

配置完后记得刷新一下

```
客户机：
ipconfig/all
ipconfig/renew

win2012：作用域停用再激活
```

配置客户保留时物理（MAC）地址为客户机的
